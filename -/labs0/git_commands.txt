git clone [url] - скопировать удаленный репозиторий на компьютер.
git status - показать состояние файлов (какие изменены, какие готовы к коммиту).
git add [file] - добавить конкретный файл в коммит. git add . добавляет все файлы.
git commit -m "message" - сохранить текущие изменения в историю с комментарием.
git push - отправить локальные коммиты на сервер (на GitHub).
git log - показать историю коммитов.
